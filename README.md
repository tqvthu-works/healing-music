### Cách sử dụng:
Khi Agent đã đọc được cả 2 file `HEALING_MUSIC_RULES.md` và `PROMPT.md`, bạn chỉ cần chat những câu rất ngắn gọn như:
* *"Cho 1 prompt Nature Hybrid"*
* *"Tạo 1 bản Lofi Healing tập trung vào tiếng mưa đêm"*
* *"Tạo 1 prompt Sleep Music với Udio"*

Agent sẽ tự động tư duy, bốc xuất ngẫu nhiên các nhạc cụ, các yếu tố tự nhiên (mưa, suối, chim hót...) và tự động mix ra một bộ prompt (gồm Style và Meta-tags) hoàn toàn mới, chuẩn xác với kỹ thuật prompting của AI!
